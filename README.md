## ✨ Features

### 🖊️ Code Editor
- Syntax highlighting for **21+ programming languages**
- Line numbers, code folding, bracket matching & auto-close
- Multi-file tabs — create, switch, rename and delete files
- Keyboard shortcuts: `Ctrl+Enter` run · `Ctrl+S` save · `Ctrl+/` comment · `Ctrl+N` new file

### ▶️ Run Code in the Browser
| Language | How it runs |
|---|---|
| **JavaScript / TypeScript** | Native browser execution with captured `console` output |
| **Python** | Skulpt runtime — real Python, loaded on demand |
| **HTML** | Live rendered in an iframe preview |
| **CSS** | Applied to a sample page and previewed |
| **JSON** | Validated and pretty-printed |
| **Markdown** | Rendered to formatted HTML preview |

### 🎨 Full Appearance Customisation
- **6 UI themes** — Dark, Darker, Nord, Monokai, Solarized, Light
- **12 editor themes** — One Dark, Dracula, Nord, Material, Monokai, Zenburn, Twilight, Tomorrow Night 80s, Night, Solarized, Eclipse, Default
- **Accent colour** — 8 presets + custom hex colour picker
- **Text colours** — individual pickers for primary, secondary and muted text
- **Editor font** — JetBrains Mono, Fira Code, Consolas, Courier New, Lucida Console, Source Code Pro
- **Font size** — slider from 10 px to 24 px
- **Line height** — slider from 1.2 to 2.4

### 💾 Auto-Save
- All files and settings are automatically saved to **`localStorage`**
- Sessions restore on next visit — zero data loss on refresh
- Manual save with `Ctrl+S`
- Auto-save can be toggled on/off in Settings

### 🗂️ File Manager
- Sidebar file explorer with language icons
- Resizable output/console panel (drag the divider)
- Split console + preview output tabs

## 🚀 Getting Started

No build step. No npm. No configuration.

1. Download `CodeForge.html`
2. Open it in any modern browser
3. Start coding

```bash
# Or clone and open directly
git clone https://github.com/your-username/codeforge.git
cd codeforge
open CodeForge.html
```
